# Contributing

Thanks for helping improve `xshow circuits`.

## Documentation Sync

This project keeps English and Chinese documentation side by side.

When updating documentation, please keep matching files in sync:

- `README.md` and `README.zh-CN.md`
- `docs/ROADMAP.md` and `docs/ROADMAP.zh-CN.md`
- English and Chinese files in `docs/issue-drafts/`
- English and Chinese issue templates in `.github/ISSUE_TEMPLATE/`

Small wording differences are fine, but the meaning, project direction, and acceptance criteria should stay aligned.

## 中文说明

感谢你一起改进 `xshow circuits`。

### 文档同步规则

这个项目会同时维护英文和中文文档。

更新文档时，请同步检查对应的中英文文件：

- `README.md` 和 `README.zh-CN.md`
- `docs/ROADMAP.md` 和 `docs/ROADMAP.zh-CN.md`
- `docs/issue-drafts/` 中的英文和中文草稿
- `.github/ISSUE_TEMPLATE/` 中的英文和中文 issue 模板

表达方式可以不完全逐字对应，但含义、项目方向和验收标准应该保持一致。
