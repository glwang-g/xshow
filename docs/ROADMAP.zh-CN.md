# 路线图

`xshow circuits` 正在朝一个友好的电子积木电路实验台推进，用来帮助用户学习和演示简单电路。

项目应该保持直观、轻量、即时反馈：用户可以搭出一个电路，看见发生了什么，并从结果里理解电路，而不需要面对完整工程仿真器的复杂度。

## 当前进度

当前重点：**导线走线和拖拽状态打磨**。

| 方向 | 状态 | 说明 |
| --- | --- | --- |
| 交互式电路实验台 | 已完成 | 电池、开关、灯泡、可变电阻、导线、亮度和回路状态已经可用。 |
| 导线选择和重接 | 进行中 | 导线可以选中、删除、拖动端点重接，也会吸附到附近端子。走线和拖拽状态还需要继续打磨。 |
| 电流方向动画 | 已完成 | 闭合回路中的通电导线已经有电流流动动画，后续还可以继续优化表现。 |
| 引导式实验课程模式 | 未开始 | 需要课程数据模型、课程面板和简单成功判断。 |

## 产品原则

- 优先使用直接操作，而不是表单配置
- 第一屏应该直接可用，而不是大段解释
- 通过动画、亮度和状态面板让电路状态可见
- 先使用适合教学的近似模型，再逐步增加物理真实度
- 保持代码规模适中，让学习者也能读懂
