# 路线图

`xshow circuits` 正在朝一个友好的电子积木电路实验台推进，用来帮助用户学习和演示简单电路。

项目应该保持直观、轻量、即时反馈：用户可以搭出一个电路，看见发生了什么，并从结果里理解电路，而不需要面对完整工程仿真器的复杂度。

## 近期方向

### 更好的导线交互

- 拖动导线端点时，自动吸附到附近端子
- 选中的导线始终位于重叠导线和端点之上
- 优化导线路径，让连接关系更容易读懂
- 补充更清晰的 hover、拖拽、释放状态

### 电流方向动画

- 当电路闭合时，在导线上显示电流流动动画
- 当电池极性变化时，动画方向也随之变化
- 当开关断开时，电流动画立即停止
- 让灯泡亮度、电流数值和导线动画之间更有视觉关联

### 引导式实验课程模式

- 增加一个课程面板，展示分步任务
- 先从电池、开关、灯泡、可变电阻的基础实验开始
- 支持简单的成功判断，比如“灯泡已经点亮”
- 课程内容应放在可编辑的数据文件中，避免写死在组件里

## 后续方向

- 更多元器件：LED、电机、蜂鸣器、二极管、电容、电流表、电压表
- 串联和并联电路示例
- 保存和加载电路工作区
- 将电路导出为图片
- 通过 URL 分享电路状态
- 在不变成完整 SPICE 仿真器的前提下，让仿真规则更真实

## 产品原则

- 优先使用直接操作，而不是表单配置
- 第一屏应该直接可用，而不是大段解释
- 通过动画、亮度和状态面板让电路状态可见
- 先使用适合教学的近似模型，再逐步增加物理真实度
- 保持代码规模适中，让学习者也能读懂
