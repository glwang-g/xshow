# 增加引导式实验课程模式

## 问题

当前实验台适合自由探索，但初学者会更需要短小、明确、有反馈的引导实验。

## 方案

增加一个课程模式，用分步任务引导用户搭建电路，并检查当前电路是否达成目标。

## 验收标准

- [x] 课程面板可以展示标题、目标和步骤
- [x] 至少提供一个“电池 + 开关 + 灯泡 + 可变电阻”的入门课程
- [x] 应用可以检测简单成功状态，比如“灯泡已经点亮”
- [x] 课程内容存储在可编辑的数据文件中，而不是写死在 UI 组件里
- [x] 用户仍然可以不进入课程模式，直接自由探索

## 备注

课程应该短小、视觉化。目标是引导发现，而不是在应用里塞一本教材。
