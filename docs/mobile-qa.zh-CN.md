# 移动端 QA 清单

本文档记录 `xshow circuits` 移动端、触控、PWA 和公开测试版发布前的手动检查项。当前清单用于 public beta / v0.4 回归测试。

## 推荐测试设备

| 场景 | 目标 |
| --- | --- |
| 手机竖屏 | 确认 HUD 不遮挡核心操作，器件抽屉和状态浮窗可用。 |
| 手机横屏 | 确认画布平移、缩放和底部 HUD 操作顺畅。 |
| iPad 横屏 | 确认触控体验接近桌面端，课程面板内容能通过状态浮窗查看。 |
| 桌面浏览器 | 确认三栏布局、鼠标拖拽、顶部按钮和属性面板没有回退。 |
| 已安装 PWA | 确认独立窗口、离线再次打开和更新检查按钮可用。 |

## 核心流程

- [ ] 打开应用后可以看到画布、紧凑任务提示浮条和底部 HUD。
- [ ] 小屏顶部没有 GitHub、复位等遮挡视线的悬浮按钮。
- [ ] 底部 HUD 中可以使用器件、状态、清线、导出、检查更新、GitHub、复位和缩放按钮。
- [ ] 点击底部器件按钮，可以呼出左侧器件抽屉。
- [ ] 点击任意器件后，器件会添加到工作台，抽屉自动收起。
- [ ] 从空白画布拖动，可以平移视图。
- [ ] 双指捏合可以缩放工作台，底部缩放比例同步变化。
- [ ] 点击“回正视图”后，手机竖屏回到上下布局的默认工作区，工作台仍适合继续操作。
- [ ] 拖动元件主体时，不会触发画布平移。
- [ ] 从端子拖到端子可以创建导线。
- [ ] 拖动导线端点可以重接到其他端子。
- [ ] 点击任务提示浮条，可以打开状态浮窗。
- [ ] 状态浮窗中可以切换实验、加载实验初始状态、查看记录。
- [ ] 点击“清线”会断开所有导线，但不会删除元器件。
- [ ] 点击“导出图片”会下载当前工作台 PNG。
- [ ] 点击“导出 JSON”会下载当前工作台存档。
- [ ] 点击“导入 JSON”可以从有效存档恢复工作台。
- [ ] 点击“复制分享链接”可以生成并复制当前工作台 URL。
- [ ] 打开分享 URL 后，可以恢复对应工作台状态。
- [ ] 添加到主屏幕后，可以以独立应用窗口打开。
- [ ] 首次联网打开后，再次离线访问可以看到应用壳或离线提示。

## 布局和旋转检查

- [ ] 手机竖屏首次打开或复位后，默认元器件按上下比例排布，导线没有明显穿过元器件主体。
- [ ] 手机横屏和 iPad 横屏复位后，工作台内容居中且留有可拖拽空间。
- [ ] 选中元器件后，可以在状态面板调整角度。
- [ ] 使用角度按钮、数字输入和滑杆调整元器件角度时，元器件不跳位。
- [ ] 旋转后的端子视觉位置和可点击/可连线热区一致。
- [ ] 旋转后的元器件参与导线连接、电路判断和 PNG 导出时保持对齐。

## PWA、更新和分享检查

- [ ] 生产环境可以安装 PWA，manifest 图标和应用名称正确。
- [ ] 已安装 PWA 中可以看到并点击“检查更新”按钮。
- [ ] 没有新版本时，检查更新按钮给出“已是最新版本”之类的短反馈。
- [ ] 发布新版本后，浏览器和已安装 PWA 能出现刷新提示，或手动检查能触发更新。
- [ ] 离线访问时，已缓存用户能看到应用壳或离线 fallback。
- [ ] 分享链接在未登录浏览器、移动端浏览器和 PWA 中都能恢复工作台。

## 云端记录检查

- [ ] 未配置 Supabase 环境变量时，云端区域显示未配置状态，本地工作台仍完整可用。
- [ ] 已配置 Supabase 时，可以注册/登录/退出。
- [ ] 登录后可以保存当前工作台到云端。
- [ ] 可以从云端记录加载、重命名和删除记录。
- [ ] 本地修改和云端记录冲突时，可以覆盖云端或另存为副本。
- [ ] 分享链接打开后，可以复制到已登录用户的云端记录。

## 课程检查

- [ ] 实验 1：点亮小灯泡可以完成所有步骤。
- [ ] 实验 2：观察断路可以完成所有步骤。
- [ ] 实验 3：调亮小灯泡可以完成所有步骤。
- [ ] 实验 4：串联两个灯泡可以识别两个灯泡、串联路径、两个灯泡点亮和串联偏暗。
- [ ] 实验 5：并联两个灯泡可以识别两个灯泡、并联支路、两个灯泡点亮和并联可见亮度。
- [ ] 实验 6：点亮 LED 可以识别正确极性、发光状态和限流保护。

## 已知限制

- 当前电路模型仍是教学近似模型，不是 SPICE 级仿真。
- 串联和并联课程的亮度比较还没有做更细的物理检查。
- 云端记录依赖生产构建时配置 Supabase 环境变量；未配置时会保持本地可用但不提供登录入口。
- 真实套件 SKU 和可打印实体装配单还没有进入 public beta 必选范围。
