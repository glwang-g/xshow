# 移动端 QA 清单

本文档记录 `xshow circuits` 移动端和触控体验的手动检查项。当前清单用于 v0.2 前的回归测试。

## 推荐测试设备

| 场景 | 目标 |
| --- | --- |
| 手机竖屏 | 确认 HUD 不遮挡核心操作，器件抽屉和状态浮窗可用。 |
| 手机横屏 | 确认画布平移、缩放和底部 HUD 操作顺畅。 |
| iPad 横屏 | 确认触控体验接近桌面端，课程面板内容能通过状态浮窗查看。 |
| 桌面浏览器 | 确认三栏布局、鼠标拖拽和顶部按钮没有回退。 |

## 核心流程

- [ ] 打开应用后可以看到画布、任务提示浮条和底部 HUD。
- [ ] 点击底部器件按钮，可以呼出左侧器件抽屉。
- [ ] 点击任意器件后，器件会添加到工作台，抽屉自动收起。
- [ ] 从空白画布拖动，可以平移视图。
- [ ] 双指捏合可以缩放工作台，底部缩放比例同步变化。
- [ ] 点击“回正视图”后，工作台回到适合继续操作的位置。
- [ ] 拖动元件主体时，不会触发画布平移。
- [ ] 从端子拖到端子可以创建导线。
- [ ] 拖动导线端点可以重接到其他端子。
- [ ] 点击任务提示浮条，可以打开状态浮窗。
- [ ] 状态浮窗中可以切换实验、加载实验初始状态、查看记录。
- [ ] 点击“清线”会断开所有导线，但不会删除元器件。
- [ ] 点击“导出图片”会下载当前工作台 PNG。
- [ ] 点击“导出 JSON”会下载当前工作台存档。
- [ ] 点击“导入 JSON”可以从有效存档恢复工作台。
- [ ] 添加到主屏幕后，可以以独立应用窗口打开。
- [ ] 首次联网打开后，再次离线访问可以看到应用壳或离线提示。

## 课程检查

- [ ] 实验 1：点亮小灯泡可以完成所有步骤。
- [ ] 实验 2：观察断路可以完成所有步骤。
- [ ] 实验 3：调亮小灯泡可以完成所有步骤。
- [ ] 实验 4：串联两个灯泡可以识别两个灯泡、串联路径和两个灯泡点亮。
- [ ] 实验 5：并联两个灯泡可以识别两个灯泡、并联支路和两个灯泡点亮。

## 已知限制

- 当前电路模型仍是教学近似模型，不是 SPICE 级仿真。
- 串联和并联课程的亮度比较还没有做更细的物理检查。
- URL 分享当前电路状态还没有实现。
