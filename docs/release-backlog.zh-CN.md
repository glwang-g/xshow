# 版本待完成清单

这份清单把当前路线图拆成可发布版本，帮助 `xshow circuits` 从“继续加功能”切换到“收口、验证、发布”。

## Public beta：公开测试版收口

状态：**收口中，先冻结大功能，只接受阻塞修复、验证记录和发布文档更新**。

当前主线已经具备公开测试版的功能面：移动端/PWA 布局、自由拖拽和任意角度旋转、课程工作台、更多元器件、节点/支路仿真、实体搭建清单、本地记录、分享链接和云端记录都已经在代码中。公开发布前，重点是把未知风险压低，而不是继续扩功能。

### 必须完成

- [x] 本地自动化检查通过：`pnpm test` 和 `pnpm build`。
- [ ] 按 [移动端 QA 清单](mobile-qa.zh-CN.md) 跑一轮手机竖屏、手机横屏、iPad 横屏和桌面回归。
- [ ] 重点复测小屏底部 HUD、默认复位布局、任意角度旋转、旋转后端子连线和 PNG 导出。
- [ ] 部署后确认线上 PWA manifest、service worker、离线再次打开和更新检查按钮可用。
- [ ] 部署后确认分享链接可以在新浏览器和移动端恢复当前工作台。
- [ ] 在线上环境验证 Supabase 登录、保存、加载、重命名、删除和冲突处理；如果暂不开放云端功能，需要在界面和文档中标明 beta/未配置状态。
- [x] 起草 [public beta release note](releases/public-beta-candidate.zh-CN.md)，说明适用场景、已知限制和反馈入口。
- [ ] 验证通过后再决定版本号、tag 和公开入口。

### 发布判断

- 没有发现会阻塞基础搭建、保存、分享、PWA 打开或课程完成的 bug。
- 未登录用户可以完整本地使用；云端不可用时不会影响工作台。
- 小屏默认布局不遮挡核心操作，复位后能直接继续编辑。
- 新版本提示或手动更新检查在远端浏览器/PWA 中可见。
