# 版本待完成清单

这份清单把当前路线图拆成可发布版本，帮助 `xshow circuits` 从“继续加功能”切换到“收口、验证、发布”。

## v0.2：移动端和分享版

状态：**Release Candidate**。

v0.2 的核心功能已经基本完成：移动端 HUD、触控平移缩放、课程任务、串并联、LED、PWA、JSON 存档、URL 分享、图片导出都已经在主线中。

### 必须完成

- [ ] 按 [移动端 QA 清单](mobile-qa.zh-CN.md) 跑一轮真机测试。
- [ ] 记录 iPad 横屏、手机竖屏、手机横屏的测试结果。
- [ ] 修复真机 QA 中发现的阻塞问题。
- [ ] 更新 README 截图或补充移动端截图。
- [ ] 确认线上部署后的 PWA manifest、service worker、分享链接都可用。
- [x] 写一份 release note。
- [ ] 真机和线上确认完成后给 v0.2 打 tag。

### 可以顺手做

- [ ] 给移动端任务浮条增加更短的文案，避免遮挡。
- [ ] 给分享链接失败、剪贴板失败补更清晰提示。
- [x] 导出 PNG 中补充当前实验标题。
- [ ] PWA 更新提示文案再口语化一点。

### 暂不塞进 v0.2

- 云端账号和自动同步。
- 更多元器件。
- 完整物理求解模型。
- 课堂/班级空间。
