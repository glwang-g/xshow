# Public Beta Candidate 发布说明草稿

状态：**草稿，等待真机 QA 和线上 PWA/云端验证后再定版本号与 tag**。

`xshow circuits` 的 public beta candidate 面向想要快速演示简单电路的教师、学生、家长和电子入门爱好者。这个版本的目标不是替代专业仿真器，而是提供一个轻量、直观、可分享的电子积木实验台：拖动元器件、连接导线、切换开关、调整电阻，然后立刻看到灯泡、LED、电机、电表和导线动画的反馈。

## 适合现在公开测试的内容

- 桌面和移动端都可用的电路工作台。
- 手机/PWA 下的底部 HUD、小屏默认复位布局和全屏画布体验。
- 电池、开关、灯泡、可变电阻、LED、二极管、电容、蜂鸣器、电机、电流表和电压表。
- 任意拖拽布局和元器件角度旋转，旋转后导线端点和 PNG 导出保持对齐。
- 基础、断路、调亮、串联、并联和 LED 六个引导实验。
- 节点/支路模型驱动的第一版教学仿真反馈。
- 本地自动保存、JSON 导入/导出、PNG 导出和分享链接。
- PWA 安装、离线再次打开和新版本更新提示/手动检查。
- 可选 Supabase 云端记录：登录、保存、加载、重命名、删除和冲突处理。
- 第一版实体搭建清单、采购关键词、接线步骤和 Markdown 导出。

## 发布前还要确认

- 手机竖屏、手机横屏、iPad 横屏和桌面浏览器回归测试。
- 远端生产环境的 manifest、service worker、离线 fallback 和更新检查按钮。
- 分享链接在新浏览器、移动端浏览器和已安装 PWA 中恢复工作台。
- Supabase 线上登录和云端记录流程；如果暂不开放云端，需要明确显示未配置/beta 状态。
- README 截图和演示入口是否与当前 UI 保持一致。

## 已知限制

- 电路模型是教学近似模型，不是 SPICE 级仿真。
- 串联/并联亮度比较已经能给教学反馈，但还不是完整物理精度。
- 云端记录依赖生产构建时配置 Supabase 环境变量；未配置时本地工作台仍可完整使用。
- 实体搭建桥接目前提供物料类型、采购关键词和接线步骤，还没有真实 SKU 映射或可打印装配单。
- 公开测试期可能会调整工作台存档格式；分享链接和 JSON 存档仍建议作为教学/演示用途使用。
