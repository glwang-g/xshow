# v0.2.0 Release Notes

Release date: 2026-05-27

## Theme

v0.2.0 is the **mobile and sharing release** for `xshow circuits`. It moves the project from a desktop-first circuit workbench toward a touch-friendly experience that works on phones and tablets, with sharing, export, PWA support, and richer guided lessons.

## Highlights

- Reworked mobile into a canvas-first HUD with a bottom toolbar, component drawer, status panel, and lesson hint strip.
- Added touch pan, pinch zoom, view recentering, and larger terminal/wire hit targets.
- Added guided series, parallel, and LED polarity lessons.
- Added lesson completion feedback.
- Added lesson checks for series and parallel brightness comparison.
- Added a polarity-sensitive LED component with reversed and over-current feedback.
- Added URL sharing for the current workbench state.
- Added JSON import/export for the current workbench.
- Added PNG export for the current workbench, including lesson title and circuit status.
- Added PWA support: manifest, home screen icons, service worker, offline fallback, and update prompt.
- Added cloud records and cross-device sync planning docs.

## Verification

- `pnpm build` passes.
- Manifest JSON validation passed in earlier work.
- Accepted for release by the project owner; follow-up device checks can continue using the [mobile QA checklist](../mobile-qa.md).

## Known Limits

- The circuit model is still an educational approximation, not a SPICE-level simulator.
- Cloud accounts and automatic cross-device sync are not implemented yet; they are planned for v0.3.
- More components and a more realistic node/branch simulation model are planned for v0.4 or later.
