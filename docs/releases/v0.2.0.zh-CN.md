# v0.2.0 发布说明

发布日期：待定

## 版本主题

v0.2.0 是 `xshow circuits` 的 **移动端和分享版**。这一版把项目从桌面优先的电路实验台，推进到手机和平板也能玩的触控体验，并补齐了分享、导出、PWA 和更完整的引导实验。

## 主要变化

- 移动端改为画布优先 HUD：底部工具条、器件抽屉、状态浮窗、任务提示浮条。
- 支持触控平移、双指缩放、回正视图，并扩大端子和导线操作热区。
- 增加串联、并联和 LED 极性实验。
- 增加实验完成反馈。
- 增加串并联亮度比较的课程检查。
- 增加有正负极的 LED 元器件，支持反接和过流提示。
- 支持复制 URL 分享当前工作台状态。
- 支持将当前工作台导入/导出为 JSON 存档。
- 支持将当前工作台导出为 PNG 图片，图片中包含当前实验标题和回路状态。
- 增加 PWA：manifest、主屏图标、service worker、离线 fallback、新版本提示。
- 增加云端记录和跨设备同步计划文档。

## 验证

- `pnpm build` 已通过。
- manifest JSON 校验已在前序改动中通过。
- v0.2 发布前仍需要按 [移动端 QA 清单](../mobile-qa.zh-CN.md) 完成真实设备检查。

## 已知限制

- 电路模型仍是教学近似，不是 SPICE 级仿真。
- 云端账号和跨设备自动同步尚未实现，已规划到 v0.3。
- 更多元器件和更真实的节点/支路模型规划到 v0.4 或之后。
